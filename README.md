NYVRA – AI-Powered Women Safety & Intelligent Route Prediction Platform

“Transforming safety from reactive to predictive using AI.”

📌 Overview

NYVRA (Navigate Your Risk Automatically) is an AI-powered women safety mobile application designed to provide proactive safety assistance through real-time risk prediction, intelligent route analysis, emergency response systems, and live location-based safety insights.

Unlike traditional safety applications that react only after danger occurs, NYVRA predicts unsafe situations before they happen using Machine Learning, geospatial analysis, heatmap visualization, and real-time route safety scoring.

The project was developed by AIML Cohort 5 as part of the IIMSTC–ECI Internship Program in collaboration with VTU.

🚀 Problem Statement

Women’s safety during travel remains a major challenge, especially during night travel and in isolated urban areas.

Most existing applications:

React only after emergencies occur
Do not provide route-level risk prediction
Ignore crime hotspot analysis
Lack intelligent AI-based safety recommendations
Provide limited emergency integration

NYVRA solves this problem through:

AI-powered danger prediction
Smart route optimization
Real-time heatmap analysis
One-tap SOS emergency support
Intelligent safety assistance
